# Telegram Bot 混合交互使用指南

## 功能概述

现在 Telegram Bot 支持三种下载方式，让你可以根据场景选择最方便的方法。

## 三种使用方式

### 方式 1: 完整命令（最快速）

适合：知道确切的账户名和帖子序号

```
/download nike 1
/dl instagram 5
```

**优点**: 一步到位，最快
**缺点**: 需要记住账户名

---

### 方式 2: 快速选择按钮（最便捷）

适合：下载常用账户的内容

**步骤**:
1. 发送 `/download`
2. 点击常用账户按钮（如 nike, instagram 等）
3. 输入帖子序号（如 `5`）

**优点**:
- 不需要输入账户名
- 避免拼写错误
- 可自定义常用账户列表

**缺点**: 需要两步操作

---

### 方式 3: 直接回复输入（最灵活）

适合：下载不在常用列表中的账户

**步骤**:
1. 发送 `/download`
2. 回复 bot 的消息，输入 `账户名 序号`（如 `nike 1`）

**优点**:
- 支持任意账户
- 不需要预设

**缺点**: 需要手动输入

---

## 配置常用账户

编辑 `config.json`，添加你常用的 Instagram 账户：

```json
{
  "telegram_bot_token": "YOUR_TOKEN",
  "allowed_user_ids": [YOUR_USER_ID],
  "favorite_accounts": ["nike", "instagram", "natgeo", "nasa", "apple", "spacex"]
}
```

**提示**:
- 可以添加任意数量的账户
- 按钮会自动排列（每行 3 个）
- 如果不配置，默认显示: nike, instagram, natgeo

---

## 高级功能

### 取消操作

如果你点击了账户按钮但想取消：
- 点击 "❌ 取消" 按钮
- 或直接发送新的 `/download` 命令

### 状态超时

如果选择账户后 5 分钟内没有输入序号，状态会自动清除，需要重新开始。

## 完整交互示例

### 示例 1: 使用按钮下载

```
你: /download

Bot: 📥 选择下载方式:

     🔹 快速选择常用账户:
     [nike] [instagram] [natgeo]
     [nasa] [apple] [spacex]

     🔹 或直接回复此消息，格式: <账户名> <帖子序号>
        示例: nike 1

你: [点击 nike]

Bot: ✅ 已选择账户: @nike

     请输入帖子序号 (例如: 1, 5, 10):
     [❌ 取消]

你: 5

Bot: ⏳ 正在下载 @nike 的第 5 个帖子...
     ✅ 下载完成！共 3 个文件 (耗时: 8.5 秒)
     正在上传...
     [图片1]
     [图片2]
     [图片3]
     ✅ 全部完成！共上传 3 个文件
```

### 示例 2: 直接回复输入

```
你: /download

Bot: 📥 选择下载方式:
     ...

你: [回复] tesla 10

Bot: ⏳ 正在下载 @tesla 的第 10 个帖子...
     ...
```

### 示例 3: 完整命令

```
你: /dl spacex 1

Bot: ⏳ 正在下载 @spacex 的第 1 个帖子...
     ...
```

## 故障排除

### 按钮没有显示

- 确保 `config.json` 中配置了 `favorite_accounts`
- 重启 bot: `./crawler bot`

### 点击按钮没反应

- 检查你的 User ID 是否在 `allowed_user_ids` 中
- 查看终端日志确认是否有错误

### 输入序号后没反应

- 确保输入的是纯数字（如 `5`，不是 `第5个`）
- 检查是否超过 5 分钟超时

---

## 技术细节

### 状态管理

Bot 会在内存中记录每个用户的当前状态：
- 选择了哪个账户
- 等待输入什么信息
- 状态创建时间

状态会在以下情况清除：
- 成功完成下载
- 用户点击取消
- 超过 5 分钟未操作
- Bot 重启

### 并发支持

多个用户可以同时使用 bot，状态互不干扰。

## 推荐使用场景

| 场景 | 推荐方式 | 原因 |
|------|---------|------|
| 经常下载 nike 的内容 | 方式 2（按钮） | 快速点击，无需输入 |
| 下载不常见的账户 | 方式 3（回复） | 灵活，支持任意账户 |
| 知道确切的账户和序号 | 方式 1（命令） | 最快，一步到位 |
| 在手机上使用 | 方式 2（按钮） | 点击比打字方便 |
| 在电脑上使用 | 方式 1（命令） | 打字快速 |
